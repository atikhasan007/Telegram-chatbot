# Gemini Telegram Chat Bot 🤖

A Telegram chatbot powered by **Google Gemini (genai)**, built with **Python** and **Aiogram**.  
Chat with an AI assistant created by Atik! 👋

---

## Prerequisites

Before you start, make sure you have:

- **Python 3.8 or higher**  
- A **Telegram account**  
- A **Google Gemini API key**  
- `conda` or `virtualenv` for environment management
