# Telegram Gemini Chatbot

## Project Overview
This project is a **Telegram chatbot** integrated with **Gemini AI** using the Google GenAI API.  
It allows users to chat with an intelligent AI model directly through Telegram, providing conversational responses in real-time.

---

## Business Objectives
- Provide a **user-friendly AI chat experience** on Telegram.  
- Automate responses to common queries and support tasks.  
- Showcase integration of **AI models with messaging platforms** for real-time interaction.  

---

## Data Sources
- **User Messages:** All input messages come directly from Telegram users.  
- **AI Model:** Responses are generated by the **Gemini-2.5-flash** model from Google GenAI API.

---

## Exploratory Data Analysis (EDA)
Since this is a **live chatbot**, traditional EDA is not applicable.  
However, message history can be logged and analyzed to improve AI responses over time.

---

## Models Used
- **Gemini-2.5-flash** (Google GenAI) – a generative AI model for conversational tasks.

---

## Evaluation Metrics
- **User Feedback:** Accuracy and relevance of AI responses.  
- **Response Latency:** Time taken to reply to a user message.  
- **Uptime and Reliability:** Continuous availability of the bot.  

---

## End-to-End Application
1. User sends a message to the Telegram bot.  
2. The bot receives the message via the **Aiogram framework**.  
3. The message is sent to **Gemini AI** for response generation.  
4. Response is sent back to the user in Telegram.  
5. Conversation is maintained per user session for context continuity.
