To follow this tutoial, you will need:

- python 3.8 or higher
- A Telegram account and a smartphone
- An OpenAI account and an API key

## Now to run the code 

1. Clone this repository or download the zip file 
2. create a virtual environment and activate it 
3. Install the dependencies using `pip install -r requirements.text`
> Or alternatively to the above two setps you can run `init_setup.sh` by running the following command in your terminal-
```batch
bash init_setup.sh
```
